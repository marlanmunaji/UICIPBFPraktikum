# Praktikum Pemrograman Berbasis Framework (PBF)

Repository ini digunakan untuk menyimpan seluruh tugas dan hasil praktikum mata kuliah Pemrograman Berbasis Framework (PBF) dari pertemuan 1 sampai pertemuan 11.

Setiap folder berisi:
- Source code praktikum
- Hasil latihan
- Penambahan komentar kode berdasarkan pemahaman pribadi
- File pendukung praktikum

## Tujuan Repository

Repository ini dibuat untuk:
- Mendokumentasikan hasil praktikum PBF
- Memahami kembali materi setiap pertemuan
- Menyimpan source code praktikum secara terstruktur
- Mempermudah proses pengumpulan tugas melalui GitHub

## Tools yang Digunakan

- Python
- Replit
- GitHub
- Visual Studio Code


## Sumber Materi

Beberapa materi praktikum berasal dari:
- https://replit.com/
- Modul dan materi pembelajaran dosen

## Catatan

Komentar pada kode program dibuat berdasarkan hasil pemahaman pribadi selama proses belajar dan praktikum berlangsung.
